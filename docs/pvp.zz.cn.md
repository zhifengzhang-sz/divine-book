### 《凡人修仙传人界篇》PVP灵书构筑真解——六链诛仙

**作者：** Z. Zhang

> 本真解构筑了一套用于修士对决的六神通位灵书体系。每一位皆为“诛仙链”上不可或缺的一环，位与位交相辉映，将整个神通的威能层层放大。此循环自成因果——第六链的终结，即是第二轮第一链爆发的开端。

### 道法总纲：乘区为基，链式诛仙

这六位一体的神通循环，乃是一套**层层递进的诛仙链**：每一链的杀机，皆根植于前一链所布下的棋局。

- **第一链**：起手定势，锋芒毕露——占据“会心”独立乘区，强攻灵力。
- **第二链**：摧枯拉朽，绝命追魂——融合“神通伤害”乘区与最大生命值斩杀，奠定胜局。
- **第三链**：改天换地，增益通天——不为瞬伤，旨在为自身施加擎天之力，为后续链节铺就无敌坦途。
- **第四链**：剥茧抽丝，终焉序曲——极致生命削减，并为下一轮循环埋下伏笔，乃跨周期之基石。
- **第五链**（推演中）：狂战引擎，以血换命——基于自损与敌之减益的真实伤害引擎。
- **第六链**（推演中）：轮回终章，万法皆空——终结此局，并完美开启下一轮更猛烈的循环。

**核心道诀**：至高伤害，源于**并行不悖的攻击路线**（以会心攻灵力，以乘法区域攻气血）与**各自独立的乘法区域**。将威能堆叠于同一加法区域，收益必将衰减。是故，每一链皆应瞄准不同的路线或乘区。

**实战为证**：乘法区域的统治力毋庸置疑，即便其数值低微。第二链中，一本近乎未经修炼的`无极御剑诀`（融合5重，悟0境）所携之【无极剑阵】（属“神通伤害”乘区），其表现亦凌驾于其他修炼圆满却仅属加法区域的词缀。故而，构筑之首务，在于占据不同的独立乘法区域；数值堆叠，反在其后。此亦道出修行真谛：提升那些能**供养核心乘法区域**的功法（例如，为“神通伤害”乘区而精进`无极御剑诀`），乃效率至高之路径。

### 斗法根本：伤害模型精析

> 完整规范参见：model/combat.md#21-the-multiplicative-damage-chain

#### 双源攻伐：气血与灵力

每位修士皆有两种本源需耗尽方可将其击败（参见../data/属性/战斗属性.md）：

| 本源 | 角色 | 攻击属性 |
| :--- | :--- | :--- |
| **气血 (HP)** | 根本生命——归零则道消 | 攻击 (ATK) |
| **灵力 (Spiritual Power)** | 消耗以生成**护盾**，格挡伤害 | 会心 (Resonance) |

此为**两条并行的攻伐路线**。多数修士专注于气血伤害（攻击/伤害乘区）。然**会心**专攻灵力——一旦灵力枯竭，对手**即无法生成护盾**，此后所有气血伤害皆可长驱直入。此乃会心价值所在：它能剥离对手的防御之甲。

#### 气血伤害——乘法区域链

神通对气血造成的伤害，将依次通过一系列**独立乘法区域**：

$$D_{神通} = \underbrace{(D_{基础} \times S_{系数} + D_{固定})}_{\text{基础伤害}} \times \underbrace{(1 + M_{伤害})}_{\text{伤害区域}} \times \underbrace{(1 + M_{神通})}_{\text{神通区域}} \times \underbrace{(1 + M_{最终})}_{\text{最终区域}} \times \underbrace{M_{心逐}}_{\text{心逐区域}}$$

| 区域 | 符号 | 来源举例 | 拥挤度 |
| :--- | :--- | :--- | :--- |
| 基础伤害 | $D_{基础}$ | 攻击力%、神通系数 | 恒定存在 |
| 伤害区域 | $M_{伤害}$ | 伤害加成、条件伤害 | **高度拥挤**——大量词缀堆叠于此 |
| 神通区域 | $M_{神通}$ | 神通伤害加成 | **极为稀缺**——来源极少（如【无极剑阵】+555%） |
| 最终区域 | $M_{最终}$ | 最终伤害加成 | **极度稀缺**——仅1-2个来源 |
| 心逐区域 | $M_{心逐}$ | 心逐效果（如【心逐神随】x2/x3/x4） | **完全独立**——作用于**所有**效果的最终外壳 |

**区域之重**：同区域内的效果彼此**相加**（收益递减）。不同区域的效果则**相乘**。在一个空白区域投入+100%，总伤即翻倍；若该区域已有+500%，再投入+100%仅能提升约17%。

#### 灵力伤害——会心 (Resonance)

会心是一条**独立的攻击路线**，目标直指灵力，而非气血伤害的乘数。

| 来源 | 效果 |
| :--- | :--- |
| 【灵犀九重】 | 必定会心，基础2.97倍，25%概率提升至3.97倍 |
| 【心逐神随】 | 乘算**所有**效果（包括会心倍率）：x2/x3/x4 |

二者兼具时：3.97 × 4 = **15.88倍灵力伤害**——足以在单次神通内抽干对手的灵力，令其在整个循环中无法生成护盾。

#### 超然通道：绕行防御之径

另有三大伤害通道，可同时绕过气血伤害链与守御（防御）：

| 通道 | 范例 | 超然之理 |
| :--- | :--- | :--- |
| **最大生命值%** | `千锋聚灵剑` 每击27% | 依目标血量而定，与自身攻击力无关 |
| **已损生命值%** | 【追神真诀】 %已损失气血 | 随目标血量降低而增长 |
| **真实伤害** | 【索心真诀】 每层减益2.1%最大生命值 | 无视所有防御 |

### 词缀择取大道

为任一链节遴选副词缀，当依此序：

**第一等：灵力攻伐 (会心)**（独立路线，战略价值至高）
- 抽干对手灵力 → 摧毁其护盾生成 → 后续所有气血伤害再无格挡。
- 来源：【灵犀九重】（必定2.97/3.97倍），可被【心逐神随】放大。

**第二等：气血乘法区域**（仅同区域内有递减）
1.  **神通区域 (M_神通)** —— 稀缺，来源极少（如【无极剑阵】+555%）
2.  **最终区域 (M_最终)** —— 极度稀缺，仅1-2来源
3.  **心逐区域 (M_心逐)** —— 独立外壳，乘算所有效果（气血与灵力）

一个能填补**空白**乘法区域的词缀，价值远胜于一个更强但与其他链节区域重复的词缀。切勿在不同链节重复占据同一乘区。

**第二等：最大生命值%/真实伤害**（绕行常规伤害链）
- 直接基于目标最大生命值百分比、已损生命值或造成真实伤害。
- 此等伤害无视多数防御性乘算——价值取决于敌之血池厚薄，而非我之攻伐高低。

**第三等：效果倍增器**（放大既有效果）
- 增益强度提升（【龙象护身】）、持续时间延长（【仙露护元】）、概率化必然（【心逐神随】）。
- 其价值完全系于所承载之“术”——单独无用，然若平台契合，则有化腐朽为神奇之效。

**第四等：加法伤害**（基础伤害、固定值、攻击力%、神通系数）
- 虽数字庞大，然彼此相加，堆叠愈高，每分收益愈薄。
- 仅在诸乘法区域皆已占据后，方虑及此。

**第五等：功能与压制**（持续伤害、减益、控制、生存）
- 如“噬心”式驱散陷阱、反治疗、无法选中状态。
- 孤木难支胜局，但可制造敌必解之困局，为我乘法伤害的爆发创造天时。

**跨链至理**：效力持久之术（增益、持续伤害、分身）在施放窗口结束后，其价值仍可绵延数链。当功法平台允可时，应优先于昙花一现之效。

**修行真谛**：资源优先倾注于那些能**供养核心乘法区域**的功法。提升一本低融合、低悟境但占据乘法区域的功法，对整体战力之助益，远胜于将一本仅提供加法效果的功法修至圆满。`无极御剑诀`在融合5重时对第二链的提升，即是明证。

**`春黎剑阵` (主位) + `解体化形`/【心逐神随】 + 【灵犀九重】 (修为)**

| 构成 | 效果 |
| :--- | :--- |

| 【心逐神随】 | 所有效果：60%概率触发4倍，80%概率触发3倍，100%概率触发2倍 |
| 【灵犀九重】 | 必定会心 (2.97倍)，25%概率提升至3.97倍 |

**道诀精微**：第一链行**双线并攻**之法：
1.  **气血线**——22305%攻击力伤害，受【心逐神随】(M_心逐 x2/x3/x4) 放大。
2.  **灵力线**——【灵犀九重】保底2.97倍会心 (25%概率→3.97倍)，受【心逐神随】放大 → 至高**15.88倍灵力伤害**。

灵力线可抽干对手灵力，摧毁其生成**护盾**之能。护盾既破，第二至第六链所有气血伤害皆可**毫无格挡**，倾泻而入。此外，分身存续16秒，其伤害受主词缀强化200%，将在后续链节中持续施压。

此剑既出，意在**一锤定音**。绝大多数对手皆陨于此链——纵有幸免者，亦已无护盾可恃。
