# 叶钦配置

> 主播叶钦的灵书配置分析。词缀详情见 [about.md](about.md)，设计约束和场景定义见 [theory.md](theory.md)。

## 组合一

| Slot | 主位 | 辅位1 | 辅位2 |
|------|------|-------|-------|
| 1 | `甲元仙符` | `元磁神光`（专属） | `浩然星灵诀`（专属） |
| 2 | `春黎剑阵` | `解体化形`（专属） | `周天星元`（【天命有归】） |
| 3 | `皓月剑诀` | `春黎剑阵`（专属） | `无极御剑诀`（专属） |
| 4 | `大罗幻诀` | `天轮魔经`（专属） | `皓月剑诀`（专属） |
| 5 | `念剑诀` | `惊蛰化龙`（专属） | `焚圣真魔咒`（专属） |
| 6 | `十方真魄` | `玄煞灵影诀`（【意坠深渊】） | `念剑诀`（专属） |

### Slot 1（增益铺垫）

- **槽位定位**：为后续槽位建立增益基础（实测 $T_{gap} \approx 6s$，12秒可覆盖 Slot 2-3）
- **词缀选取与目标交付**：
  - `甲元仙符`主位 → 【天光虹露】：释放时获得【仙佑】（+70%攻击/守御/气血，12秒），为后续槽位建立增益基础
  - `元磁神光`辅位 → 目标【真极穿空】：增益层数+100%，每5层+5.5%伤害 → 放大【仙佑】的叠层收益
  - `浩然星灵诀`辅位 → 目标【龙象护身】：增益效果强度+104% → 直接翻倍【仙佑】的属性加成
- **协同**：【龙象护身】放大增益幅度，【真极穿空】放大增益层数 → 乘法增强
- **跨槽价值**：【仙佑】增益持续12秒，实测 $T_{gap} \approx 6s$，可覆盖 Slot 2（6s）和 Slot 3（12s临界）。Slot 4 及之后不在覆盖范围内（18s > 12s）

### Slot 2（伤害倍增）

- **槽位定位**：分身 + 确定性高倍乘数
- **词缀选取与目标交付**：
  - `春黎剑阵`主位 → 【幻象剑灵】：分身伤害+200%，存在16秒，分身会跟随施放后续神通
  - `解体化形`辅位 → 目标【心逐神随】：11%×4倍 + 31%×3倍 + 51%×2倍（期望×2.46）→ 核心倍增词缀
  - `周天星元`辅位 → 目标【天命有归】：概率触发→必定触发 + 50%伤害 → 将心逐从期望×2.46提升至确定×4.00
- **组合乘数**：×4.00 × ×1.50 = **×6.00**
- **跨槽价值**：分身持续16秒，$T_{gap} \approx 6s$，可跟随 Slot 3（6s）和 Slot 4（12s），Slot 5 临界（16s vs 18s，不可覆盖）；本槽直接伤害 22,305% × 6.00 = 133,830%
- **关键**：【天命有归】消除【心逐神随】的随机性，将高风险高收益转化为确定性峰值输出

### Slot 3（主攻输出）

- **槽位定位**：最高段数爆发 + 护盾破坏 + %maxHP伤害
- **词缀选取与目标交付**：
  - `皓月剑诀`主位 → 【碎魂剑意】：基于湮灭护盾数量每0.5秒造成高额伤害 → 最高段数（10段）、护盾破坏 + %maxHP伤害
  - `春黎剑阵`辅位 → 目标【玄心剑魄】：附加【噬心】DoT（550%/秒，8秒），驱散则3300%爆发+2秒眩晕 → 为目标制造"忍受DoT还是驱散受罚"的两难
  - `无极御剑诀`辅位 → 目标【无极剑阵】：神通伤害+555%（代价：目标+350%神通伤害减免）→ 本槽最大化单次伤害
- **跨类型复用**：`春黎剑阵` = Slot 2 主位 / Slot 3 辅位 → 无冲突
- **协同**：Slot 2 分身跟随 Slot 3 攻击 → 分身继承54%属性、伤害+200%，为 Slot 3 提供额外输出

### Slot 4（辅助输出）

- **槽位定位**：反击诅咒 + 【命损】前置 + DoT递增循环
- **词缀选取与目标交付**：
  - `大罗幻诀`主位 → 【魔魂咒界】：自身获得【罗天魔咒】（被攻击时60%概率给攻击方叠加【噬心之咒】/【断魂之咒】），额外施加【命损】（最终伤害减免-100%，8秒）
  - `天轮魔经`辅位 → 目标【心魔惑言】：减益层数+100% → 若作用于【罗天魔咒】的【噬心之咒】/【断魂之咒】叠层，可翻倍叠层速度（待验证：【心魔惑言】是否适用于被攻击触发的减益）
  - `皓月剑诀`辅位 → 目标【追神真诀】：持续伤害触发时+26.5%已损失HP伤害；悟10境：+50% maxHP伤害、+300%总伤害 → 将DoT层数转化为爆发性HP%伤害
- **跨类型复用**：`皓月剑诀` = Slot 3 主位 / Slot 4 辅位 → 无冲突
- **协同**：若【心魔惑言】生效 → 减益层数增加 → 【噬心之咒】/【断魂之咒】DoT更多 → HP损失更大 → 【追神真诀】按已损失HP比例造成更高伤害 → 形成递增循环（前提：【心魔惑言】适用于被攻击触发的减益，待验证）
- **跨槽价值**：【命损】持续8秒，$T_{gap} \approx 6s$，仅可覆盖1个后续槽位（Slot 5）。Slot 6（12s）超出覆盖范围

### Slot 5（持续消耗）

- **槽位定位**：持续DPS + 减益变现 + DoT频率翻倍
- **词缀选取与目标交付**：
  - `念剑诀`主位 → 【雷阵剑影】：技能结束后雷阵额外持续6.5秒，每0.5秒一次伤害 → 提供持续DPS
  - `惊蛰化龙`辅位 → 目标【紫心真诀】：目标每有1层减益+2.1% maxHP真实伤害（最多21%）→ 将 Slot 2–4 累积的减益层数变现为HP%真伤
  - `焚圣真魔咒`辅位 → 目标【天魔真解】：持续伤害触发间隔-50.5% → 本槽位DoT（雷阵剑影）的tick频率翻倍
- **协同**：【天魔真解】使雷阵从0.5秒/次 → ~0.25秒/次；【紫心真诀】触发条件为"本神通造成伤害时"，若DoT加速后的每次伤害均计为触发，则触发频率也相应提升（待验证）
- **【命损】加持**：Slot 4【命损】持续8秒，到 Slot 5（6秒后）仍在持续，所有伤害进一步放大。但到 Slot 6（12秒后）已过期

### Slot 6（收尾保障）

- **槽位定位**：自保续航 — 提供持久的攻击/减伤加成，掩护前序 DoT 和分身完成击杀
- **词缀选取与目标交付**：
  - `十方真魄`主位 → 【星猿弃天】：【怒灵降世】延长3.5秒（4→7.5秒）+ 每秒30%驱散自身控制
  - `玄煞灵影诀`辅位 → 目标【意坠深渊】（体修修为词缀）：自身已损气血值计算伤害时至少按已损11%计算，并使本神通伤害+50% → 为`十方真魄`的已损气血伤害提供保底（即使满血也按11%已损计算），同时无条件+50%伤害
  - `念剑诀`辅位 → 目标【仙露护元】：增益状态持续时间+300% → 延长【怒灵降世】（+300%作用于基础4秒还是【星猿弃天】后的7.5秒存在歧义：前者→16秒，后者→30秒，待验证）
- **跨类型复用**：`念剑诀` = Slot 5 主位 / Slot 6 辅位 → 无冲突
- **协同**：`十方真魄`消耗10%当前HP → 触发已损气血伤害机制，且【意坠深渊】保证至少按11%已损计算（即使释放前近满血），+50%无条件伤害进一步放大【怒灵降世】期间的输出 → 【仙露护元】延长【怒灵降世】→ 在先前DoT和分身收尾期间维持攻防加成

### 组合一冲突分析

组合一有3处跨类型复用，均为一处主位、另一处辅位，不触发冲突：
- `春黎剑阵`：Slot 2 主位 / Slot 3 辅位
- `皓月剑诀`：Slot 3 主位 / Slot 4 辅位
- `念剑诀`：Slot 5 主位 / Slot 6 辅位

**结论：组合一无冲突**

### 组合一实测观察

#### 实验条件

- Slot 1-3：已构造灵书，但 Slot 3 辅位2 使用`念剑诀`替代`无极御剑诀`（无极尚未获得）
- **Slot 4-6：未构造灵书，使用基础功法**（伤害远弱于完整灵书）
- 对手：PvP 环境

#### 观察结果

1. **将原 Slot 3 配置（皓月）移至 Slot 1 后，皓月的输出显著提升**
2. **Slot 2（春黎）的输出远超 Slot 3（皓月）**。春黎在 Slot 2 拥有 ×6.00 组合乘数（【天命有归】×【心逐神随】），理论直接伤害 133,830%，实际伤害表现也远高于 Slot 3 的皓月。这意味着在当前配置下，Slot 2 才是事实上的主攻位，而非理论框架中假设的 Slot 3
3. **`念剑诀`辅位取【心火淬锋】（剑修修为）远优于【仙露护元】（念剑诀专属）**。`念剑诀`作为 Slot 3（皓月）的辅位2 时，【心火淬锋】的实际表现远好于【仙露护元】

#### 互换后的排列

| Slot | 主位 | 辅位1 | 辅位2 | 变化 |
|------|------|-------|-------|------|
| 1 | `皓月剑诀` | `春黎剑阵`（专属） | `念剑诀`（替代无极） | ← 原 Slot 3 |
| 2 | `春黎剑阵` | `解体化形`（专属） | `周天星元`（【天命有归】） | 不变 |
| 3 | `甲元仙符` | `元磁神光`（专属） | `浩然星灵诀`（专属） | ← 原 Slot 1 |
| 4-6 | （基础功法，未构造灵书） | | | 不变 |

#### 可能的解释

**猜想A：增益边际收益递减**。`甲元仙符`的【仙佑】（+70%攻击力加成）对 Slot 2（春黎，已有 ×6.00 内在乘数）的边际增幅相对较小。将【仙佑】推迟至 Slot 3 释放，覆盖 Slot 4-5（$T_{gap} \approx 6s$，12秒可覆盖2个后续槽位），对弱槽位的增幅反而更大。但此猜想需要终态验证——当 Slot 4-6 构建完整灵书后，【仙佑】覆盖 Slot 2-3 vs Slot 4-5 的收益对比可能改变

**猜想B：Slot 4-6 弱势放大了互换效果**。当前 Slot 4-6 使用基础功法，伤害远低于完整灵书。【仙佑】覆盖这些弱槽位时的**绝对增幅**（70% × 低基础）可能仍高于覆盖已经很强的 Slot 2 的**边际增幅**。这是一个过渡期现象，在 Slot 4-6 构建完成后可能不再成立

**猜想C：皓月的独立输出特性**。`皓月剑诀`的伤害包含 12%maxHP/段（不依赖攻击力）和【碎魂剑意】的护盾机制伤害，这些%HP伤害不受【仙佑】的攻击力加成影响。因此皓月放在 Slot 1（无增益）的损失小于预期

**猜想D：前置输出争取战斗时间**。皓月放在 Slot 1（0秒）比 Slot 3（12秒）早12秒造成伤害。在PvP中，更早的高额伤害意味着更多战斗时间优势——对手更早承压，减少其反应和回复的窗口。尤其当 Slot 2（春黎）的 ×6.00 乘数是自身完备的（【天命有归】×【心逐神随】为槽位内效果），不依赖前序增益铺垫，Slot 1 的增益铺垫对 Slot 2 的边际贡献有限，不如直接前置一个高伤害槽位

**猜想E：缺少无极的影响**。实测中 Slot 3 辅位2 使用`念剑诀`替代`无极御剑诀`（未获得），因此皓月缺少【无极剑阵】的+555%神通伤害加成。获得无极后皓月在 Slot 3 的输出可能显著提升，互换效果可能减弱

#### 观察3分析：【心火淬锋】vs【仙露护元】

`念剑诀`作为`皓月剑诀`辅位时，可能获得的词缀包括：
- 【仙露护元】（念剑诀专属）：使本神通添加的`增益`状态持续时间延长300%
- 【心火淬锋】（剑修修为）：本神通命中时，每造成1段伤害，剩余段数伤害提升5%，最多提升50%

`皓月剑诀`主技能为10段攻击，并给自身添加【寂灭剑心】（持续4秒）。

**【心火淬锋】对皓月的收益**：10段逐段递增，Hit 1 ×1.00, Hit 2 ×1.05, ... Hit 10 ×1.45，平均 ×1.225，即**+22.5%总伤害**，且后段增幅更大（与皓月自身的段数递进机制叠加）。

**【仙露护元】对皓月的收益**：延长【寂灭剑心】从4秒到16秒。但皓月的10段攻击在技能施放期间即可全部完成（远短于4秒），额外的12秒延长期间没有后续攻击段数可触发，**实际收益接近零**。

**关键认知：辅位词缀应以与主位技能的协同为第一优先，而非词缀稀缺性**。专属词缀的出现概率最低（专属 < 修为 < 通用），在灵书构造过程中往往被视为最优目标。但专属词缀是为其**来源功法**设计的——【仙露护元】服务于`念剑诀`自身的雷阵增益延长，放在`皓月剑诀`上则缺乏协同目标。反之，修为词缀【心火淬锋】虽然稀缺性较低，但其逐段递增机制与皓月的10段攻击天然匹配，实际收益远高于专属。灵书构造时应先分析主位技能的伤害结构（段数、%HP、增益依赖等），再从辅位功法的全部词缀池中选取协同最优的目标，不应默认专属 > 修为 > 通用

#### 延伸：【仙露护元】的最优主位

既然【仙露护元】（+300%增益持续时间）不适合`皓月剑诀`，那么哪个主位才能最大化其价值？关键条件：主位技能需添加**高价值增益状态**，且**持续时间是覆盖瓶颈**。

| 主位 | 增益状态 | 基础时长 | +300%后 | 协同分析 |
|------|---------|---------|---------|---------|
| **`甲元仙符`** | **【仙佑】+70%攻击/守御/气血** | **12秒** | **48秒** | **覆盖从2个槽位→全部5个后续槽位（$T_{release} \approx 30s$）。直接解决 buff 覆盖不足的核心问题** |
| `春黎剑阵` | 分身（若算增益状态） | 16秒 | 64秒 | 待验证分身是否属于增益状态 |
| `十方真魄` | 【怒灵降世】+20%攻击+减伤 | 4秒（+星猿3.5s=7.5s） | 16秒/30秒 | 可观，组合一 Slot 6 已在用此组合 |
| `大罗幻诀` | 【罗天魔咒】反击触发 | 8秒 | 32秒 | 更长的反击窗口，更多【命损】触发机会 |

**最优主位：`甲元仙符`**。当前配置的核心痛点是【仙佑】（12秒）仅能覆盖2个后续槽位（$T_{gap} \approx 6s$）。若`念剑诀`作为`甲元仙符`的辅位取得【仙露护元】，【仙佑】延长至48秒，足以覆盖全部后续槽位，彻底消除 buff 覆盖瓶颈。

当前组合一的 Slot 1（`甲元仙符`）辅位为 `元磁神光` + `浩然星灵诀`，目标是【真极穿空】+【龙象护身】来放大增益强度。若将其中一个辅位替换为`念剑诀`（目标【仙露护元】），则是**增益强度 vs 增益覆盖**的取舍：

- 当前方案：强增益（×2.04强度 + ×1.275层数），短覆盖（2个槽位）
- 替换方案：较弱增益（失去一个放大词缀），全覆盖（5个槽位）

这一取舍需要实测验证，但从理论上看，全覆盖的总收益很可能更高——即使单槽增幅降低，5个槽位受益 vs 2个槽位受益的差距是决定性的。

### Slot 2（伤害倍增 + 【命损】前置）— 核心结构性差异

- **槽位定位**：【命损】前置 + 减益铺垫（牺牲确定性倍增，换取更广的【命损】覆盖窗口）
- **词缀选取与目标交付**：
  - `大罗幻诀`主位 → 【魔魂咒界】→ 【命损】（-100%最终伤害减免，8秒）**前置至 Slot 2**
  - `解体化形`辅位 → 目标【心逐神随】：期望×2.46（本组合无【天命有归】，不再是确定×4.00）
  - `天轮魔经`辅位 → 目标【心魔惑言】：减益层数+100% → 若适用于【罗天魔咒】的被攻击触发减益，可强化叠层效率（待验证）
- **关键取舍**：牺牲【天命有归】的确定×4.00（Slot 2 直接伤害从133,830%降至49,852%），换取【命损】从 Slot 2 开始覆盖。但实测 $T_{gap} \approx 6s$，命损（8秒）仅能覆盖**1个后续槽位**（Slot 3）。相比组合一 Slot 4→覆盖 Slot 5，组合二覆盖的是 Slot 3（主攻位），该槽位伤害更高，命损收益因此更大
- **跨槽价值**：【命损】覆盖 Slot 3（主攻位），在高DR场景下放大效果显著（50%DR→×2，70%DR→×3.33，90%DR→×10），但仅覆盖1个槽位。

### Slot 3（主攻输出）

与组合一相同配置，但现在受益于 Slot 2【命损】（Slot 2 → Slot 3 间隔6秒，在8秒覆盖内）：
- 【无极剑阵】的+350% DR惩罚在【命损】（-100% DR）下可能被完全抵消
- 【命损】使【碎魂剑意】的DoT和`皓月剑诀`本身的%HP伤害均获得大幅增益
- **注意**：【命损】到 Slot 4 时已过期（12s > 8s），仅 Slot 3 受益

### Slot 4（辅助输出）— 新组合，高爆发设计

- **槽位定位**：递增爆发 + HP%伤害 + 无条件高倍乘数
- **词缀选取与目标交付**：
  - `千锋聚灵剑`主位 → 【惊神剑光】：每段伤害后下一段+25%神通加成 → 6段递增
  - `皓月剑诀`辅位 → 目标【追神真诀】：持续伤害触发时+26.5%已损失HP；悟10：+50% maxHP，+300%总伤害 → 放大`千锋聚灵剑`的每段15% maxHP伤害
  - `十方真魄`辅位 → 目标【破釜沉舟】：伤害+380%（×4.80），代价：自身受伤+50% → 无条件高倍乘数
- **跨类型复用**：`皓月剑诀` = Slot 3 主位 / Slot 4 辅位 → 无冲突
- **协同**：【破釜沉舟】×4.80 × 【惊神剑光】递增（平均约×1.625）× 【追神真诀】HP%伤害（若触发） → 多重乘数叠加（各乘数间是否为乘法关系待验证）。Slot 2【命损】到 Slot 4 已过期（12s > 8s），Slot 4 **不受命损加持**
- **待验证**：`千锋聚灵剑`的15% maxHP伤害是否算作"持续伤害"以触发【追神真诀】

### Slot 5（持续消耗）— 分身续航设计

- **槽位定位**：分身续航 + 增益延长
- **词缀选取与目标交付**：
  - `春黎剑阵`主位 → 【幻象剑灵】：分身伤害+200%，存在16秒
  - `念剑诀`辅位 → 目标【仙露护元】：增益状态持续时间+300% → 若分身算作"增益状态"则可大幅延长分身持续时间（待验证：分身是否属于增益状态）
  - `大罗幻诀`辅位 → 目标：【业焰】（状态延长+69%）/ 【清灵】（增益强度+20%）/ 【古魔之魂】（DoT+104%）→ 三种目标均服务于分身续航策略
- **跨类型复用**：`春黎剑阵` = Slot 3 辅位 / Slot 5 主位；`大罗幻诀` = Slot 2 主位 / Slot 5 辅位 → 均为跨类型，无冲突
- **【命损】加持**：Slot 2【命损】到 Slot 5 已过期（18s > 8s），分身攻击**不受命损加持**

### Slot 6（收尾保障）— 延迟爆发设计

- **槽位定位**：延迟爆发 — 收集先前所有伤害，最终一次性结算
- **词缀选取与目标交付**：
  - `无相魔劫咒`主位 → 【灭劫魔威】：【无相魔劫】持续12秒，期间敌方神通伤害+10%，结束时造成（期间增伤的10% + 5000%攻击力）的伤害；【灭劫魔威】使结算伤害+65%
  - `疾风九变`辅位 → 目标【真言不灭】（体修专属）：所有状态持续时间+55% → 将【无相魔劫】收集窗口从12秒延长至18.6秒，使更多前序DoT和分身伤害喂入最终结算（Slot 5 分身16秒，在18.6秒窗口内可被完整收集）
  - `周天星元`辅位 → 目标【奇能诡道】：减益添加时20%概率多叠1层；悟境条件下施加【逆转阴阳】（降低敌方DR）→ 增强`无相魔劫咒`的减益效果
- **协同**：【真言不灭】延长收集窗口（12s→18.6s），使 Slot 5 分身攻击被完整收集，【奇能诡道】增加减益叠层效率。Slot 2【命损】到 Slot 6 已过期（24s > 8s），**不受命损加持**
- **注**：`无相魔劫咒`至少悟1境，不再需要【天人合一】补偿悟境

### 组合二冲突分析

装配规则第1条为总则，第2、3条为总则的完整细则：
- `核心冲突`：同一功法在两个槽位均为**主位** → 高位主技能失效
- `副词缀冲突`：同一功法在两个槽位均为**辅位** → 高位词缀失效

组合二中有3处同一功法出现在两个槽位，但均为**跨类型**（一处主位、另一处辅位），不触发上述任一冲突规则：
- `皓月剑诀`：Slot 3 主位 / Slot 4 辅位 → 皓月未在两个槽位均为主位或均为辅位，无冲突
- `大罗幻诀`：Slot 2 主位 / Slot 5 辅位 → 同理，无冲突
- `春黎剑阵`：Slot 3 辅位 / Slot 5 主位 → 同理，无冲突

**结论：组合二无冲突**

## 疑问

1. `本神通`是指灵书的所有神通？还是副书本身的神通
   - 分析结论：根据词缀描述措辞（"本次神通所有效果"等），`本神通`应指所在槽位的灵书技能，不carry到其他槽位。只有明确描述跨槽的效果（如【灵威】"使下一个施放的神通"）才会影响后续槽位
2. 【天命有归】能否使【心逐神随】的概率效果提升为必定触发（始终4倍）？
   - **已确认：可以**。【天命有归】使【心逐神随】的概率效果确定触发最高倍率（×4.00），组合后 $M_{affix}$ = ×4.00 × ×1.50 = ×6.00
3. `甲元仙符`主技能【仙佑】（+70%攻击/守御/气血）持续12秒，是否能覆盖到 Slot 3？
   - **已解决**：实测 $T_{gap} \approx 6s$，Slot 1→Slot 3 间隔12秒，恰好临界。Slot 2（6s）确定覆盖，Slot 3（12s）临界，Slot 4 及之后（18s+）不在覆盖范围内
4. 跨类型复用（同一功法在一个槽位为主位、另一个为辅位）不触发冲突，因为不满足核心冲突（需同一功法在两个主位）或副词缀冲突（需同一功法在两个辅位）的条件。这意味着功法可以在不同槽位以不同角色（主位/辅位）出现而不受限制
5. `追神真诀`Part 1（持续伤害触发时+26.5%已损HP）在千锋聚灵剑上是否能触发？千锋的15%maxHP/段为直接伤害而非DoT，可能不满足"持续伤害"条件
6. 【命损】覆盖窗口：【罗天魔咒】持续8秒，期间被攻击60%概率触发【命损】（持续8秒）。Slot 2 释放后到 Slot 6 释放完毕需多长时间？
   - **已解决**：实测 $T_{gap} \approx 6s$，Slot 2→Slot 6 间隔 4×6s=24s，远超8秒。【命损】仅能覆盖1个后续槽位：组合二中 Slot 2→覆盖 Slot 3；组合一中 Slot 4→覆盖 Slot 5。"【命损】覆盖所有后续槽位"的假设不成立
